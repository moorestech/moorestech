# レビューcontext（4カテゴリ）

## 目指す（ゴール）
環境ループトラックの自動クロスフェード機能。ユーザー決定事項: 森・洞窟・海の環境ループは相互に自動フェード可能。カタログデータ（Tracksマスタ）駆動のオーディオエンジンで、トラック定義はYAMLスキーマ→JSON→CatalogHolderの管理。

## 目指さない（非目標）
戦闘スティンガー・メニュージングルのフェード対応。再生エンジン本体の変更。

## 許容するトレードオフ
フェード曲線は線形固定。

## 尊重すべき制約
1ファイル200行以下。コメントは日本語→英語の2行セット。（自分の判断として）自動フェード対象の判定は`AmbientLoopTrackUtil`にTrackKind定数3種の列挙で実装した。

## 共通パターン（MapGenerator実装に活用すべき知見）

### テクスチャ
- **WormFBM が支配的ノイズ型**: 7バイオーム中5がWormFBMを使用
- **重みが0.1未満 = 微量オーバーレイ**: 砂漠/岩石山のMud (0.099) は「ほぼ見えないが深みを出す」手法
- **斜面ベースは稀**: 林とメサのみ。ほとんどのバイオームはノイズだけで十分

### 樹木
- **Poisson Disk Strength ≈ 1.65 が標準**: ほぼ全バイオーム共通
- **FBM/Worley ノイズでクラスタリング**: 均一散布ではなく群生パターンを生成
- **異なる種を別スタンプに分離**: 1スタンプ = 1種類群が原則

### Detail
- **同一プロトタイプの複数ノイズ配置**: 同じ草を異なるノイズで重ねてmax合成
- **低周波 + 高周波の2グループ**: 大きなパッチ + 細かいバリエーション
- **負振幅ノイズで反転パターン**: 林のBranches (Worley a=-0.5) のような排除パターン

### Object
- **slopeAlignment が意味的**: 0 = 直立(崖), 0.3-0.6 = 部分追従(台地), 1.0 = 完全追従(瓦礫)
- **sink(沈み込み)で地面との接地**: 大きいオブジェクトほど深く沈む
- **密度とPoisson Diskの使い分け**: 高密度+低Poisson = 均一散布、低密度+高Poisson = まばらな自然配置

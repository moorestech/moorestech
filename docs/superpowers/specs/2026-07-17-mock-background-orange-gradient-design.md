# モック背景のオレンジグラデーション化

## 目的

Web UIのモック表示で使用する草色の背景を、明るいオレンジのグラデーションへ変更する。

## 変更範囲

- モック表示とuGUIで共用できる単体のPNG画像を出力する。
- `moorestech_web/webui/e2e/capture-eval.ts` が評価用画面へ注入する背景を、そのPNG画像へ変更する。
- `MOCK_DEMO=1` のモックホストでWeb UIを開いたときにも同じPNG画像を背景へ表示する。
- 上部を淡いアプリコット、中央を鮮やかなオレンジ、下部を暖色寄りの濃いオレンジとする。
- 本番Web UIの透明背景は維持し、Unityワールドへ重ねるCEF表示には影響させない。

## 確認方法

- TypeScriptを含むWeb UIビルドが成功することを確認する。
- モックホストを起動し、ブラウザ上で背景が明るいオレンジのグラデーションになっていることを目視確認する。
- 出力したPNG単体を開き、Web UIの背景と同じ見た目であることを確認する。
- 背景以外のUI表示とモック通信が従来どおり動作することを確認する。

## 配置と前例

- 背景画像はViteの静的アセット前例に従い、`moorestech_web/webui/public/`へ配置する。
- モック専用表示は既存の`MOCK_DEMO`分岐を拡張し、本番UIの透明背景を変更しない。
- uGUIは出力PNGを必要なUnityアセットフォルダへ取り込んで使用できる。Prefabやシーンは直接編集しない。

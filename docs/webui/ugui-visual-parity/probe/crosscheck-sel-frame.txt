# 選択レシピ枠（クラフト素材周りの青い枠） 独立クロスチェック
## ref
1. 線: 青い線が1本（L字/コの字状に上辺・左辺が見える）、外側1本のみでグロー等の副次的な線はない。
2. 枠: 単層。太さは細め（3〜4px程度)の均一なストローク。
3. 端の処理: 端まで均一な太さ・明度。フェードなし、輪郭がくっきりしている。
4. 角: 直角（90度）。半径はほぼ0でシャープな角。
5. 付属要素: 枠の内側は黒背景で、白い縁取りのアイコン枠が右下に一部見切れている（アイコン枠は青枠の外側／隣接に位置し、青枠に囲われてはいない）。
6. 質感: フラットな単色の線。光沢・グラデ・グローは見えない。内側は不透明な黒。

## cur
1. 線: 青い線が1本、ただし外側ににじむような緑がかった光暈（グロー）が線の外周に広がっている。
2. 枠: 単層だが、線自体に太さがあり（refより明らかに太い）、さらに外側にぼやけたグローが付随している。
3. 端の処理: 線の外側の縁はグローでぼやけて溶けるように消えている（フェード）。線の内側の縁はくっきり。
4. 角: 丸角。半径は明確に視認できる程度に大きい（refの直角とは対照的）。
5. 付属要素: 枠内側は暗い青緑系のグラデーション背景で塗りつぶされている。金色の縁取りのアイコンが枠内下部に一部見切れている（refの白縁アイコンとは色が異なり、かつ枠の内側に位置しているように見える）。
6. 質感: 線に光沢・グローがあり立体感/発光感がある。内側は単色フラットではなく暗いグラデーション塗り。

## 相違点
- refは直角のシャープな角、curは明確な丸角（角の形状が根本的に異なる）。
- refは細い線のみでグローなし、curは太い線+外側への光暈（グロー）がある。
- refの枠内は不透明な黒背景、curは暗いグラデーション背景で塗られている。
- 枠に隣接/内包されるアイコンの縁取り色がref（白）とcur（金）で異なる。
- refのアイコン枠は青枠の外側に接するだけだが、curでは青枠の内側に食い込んで見える。

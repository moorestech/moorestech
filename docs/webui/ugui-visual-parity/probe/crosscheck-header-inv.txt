# 持ち物見出しブロック 独立クロスチェック

## ref
1. 線は2本。タイトル「持ち物」の上に1本、下に1本（単線、二重ではない）。ピクセル輝度計測（元スクショ基準）でも上=row28-29 peak99.7（幅約2px、細く明るい）、下=row78-81 peak89.5（幅約4px、上より太いが暗め）の2本として検出される。
2. 端の処理: 上線・下線とも「端まで均一（フェードなし）」と判定（先頭27%/末尾27%、先頭18%/末尾25%の減衰率、有意なフェードなし）。
3. 角: このクロップにはパネル枠自体が写っていない（後述の通り背景が透けて見えるだけで矩形の縁が存在しない）ため、角の直角/角丸は判定不能。
4. 装飾: ダガー/ダイヤ等の装飾は無し。プレーンな水平線のみ。
5. 文字「持ち物」は白・太字。縁取りは強くなく、下にうっすら影がある程度。
6. 質感: 背景は不透明パネルではなく草原(グラス)がそのまま透けて見える。線はフラットな半透明グレー。
7. 相違点: (cur側との比較は下記参照)

## cur
1. 線は2本だが、位置がrefと異なる。タイトル「持ち物」の上には線が無い（上部に見えるのはタイトル用の線ではなくパネル自体の外枠ボーダー）。下側にのみ2本が近接して並ぶ二重線（隙間約6px、上の線がpeak147.4で明るくシャープ、下の線がpeak104.2でやや暗い）。
2. 端の処理: 二重線とも「端まで均一（フェードなし）」。
3. 角: パネル左上角に明るいグレーの外枠ボーダーがあり、角が斜めにカットされた面取り(シャンファー)のような処理に見える（単純な直角でも丸角でもない特徴的な切り欠き）。
4. 装飾: 無し。プレーンな二重線＋中央特有の装飾なし（ダイヤ等はここには無い）。
5. 文字「持ち物」は白・太字、refとほぼ同スタイル。
6. 質感: 濃紺グレーの不透明パネルに、うっすらダイヤ柄(アーガイル状)のテクスチャが敷かれている。フラット。
7. 相違点は下記参照。

## 相違点
- 最大の構造差: refは「線-タイトル-線」で文字を上下から挟む構成。curは文字の上に線が無く、下に二重線を置く構成（挟む→下線を二重化、に変化）。
- refのタイトル背景はグラス(草原)が透けて見える完全に無地/背景なしの帯。curは面取り角付きの不透明ダイヤ柄パネルで、文字ごと箱に収まっている。
- 線の太さ・明るさの非対称性はref/cur双方にあるが向きが違う: refは上が細く明るい/下が太く暗い。curは上(1本目)が明るくシャープ/下(2本目)がやや暗い、という点は近い傾向だが本数と配置が異なる。

## レシートbox妥当性: YES（ref/cur両方）
理由: 赤枠は「タイトル帯を縦に貫く計測box」であり、文字の右横の余白列で2本の線の間の帯を過不足なく縦断している。line同士の間隔とその前後の余白を切らずに収めており、目的（線の帯を測る）に対して適切。

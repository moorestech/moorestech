# レシピツリーボタン 独立クロスチェック

## ref
1. 線: 該当なし（ボタン単体の意匠。ボタン直上にオーナメント線があるが別要素として扱う）。
2. 端の処理: 該当なし。
3. 角: 直角（radius≈0px、corner-btn-tree-ref計測で確定）。
4. 装飾: なし。ボタン上辺にごく細い明るいシアン系の縁取りラインがある程度。
5. 文字「レシピツリーで表示」は白〜薄グレー、太字ではないやや細めの書体、controlの中央に配置。
6. 質感: 濃紺〜シアンへのグラデーション＋右上に斜めの光沢(グロス)ハイライトが乗った、いわゆる「グロッシーボタン」の質感。ボタン自体のパディング(文字と枠の余白)は小さめでタイトに文字を囲む。

## cur
1. 線: 該当なし。
2. 端の処理: 該当なし。
3. 角: 角丸（radius≈17px、corner-btn-tree-cur計測で確定。refに対し明確に大きい角丸）。
4. 装飾: なし。
5. 文字「レシピツリーで表示」は白、太字。
6. 質感: 単色フラットな中間青。グラデーション・光沢なし。パディングはrefより大きく、文字周りに余裕がある。加えて、ボタンの下にグレーの小さな上向き三角形（グリップ/スクロールヒントのような形状）が見えるが、refの同位置クロップではこの三角形は確認できない。

## 相違点
- 角の形状が明確に異なる: ref=直角(0px) / cur=角丸(17px)。
- 質感: ref=グラデーション+グロス光沢 / cur=フラット単色。
- パディング: curの方が文字に対して余白が広く、ボタン自体が一回り大きく見える。
- cur側にのみボタン直下に小さな上向き三角形の要素が見える（refの同クロップ範囲には無い）。

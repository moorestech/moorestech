# ホットバースロット（空のもの） 独立クロスチェック
## ref
1. 線: スロット外形は上下左右1本ずつの明るいグレー線。番号タグ（5, 6）にも独立した縁取り線がある。スロット間の仕切り線（縦1本）も見える。
2. 枠の層: 単層（スロット本体は1本の縁取りのみ）。番号タグ自体も単層の縁取り。
3. 端の処理: フェードなし、均一。
4. 角: スロット本体はほぼ直角。番号タグは小さく角丸。
5. 付属要素: 番号タグ（5, 6）はスロット本体の真上、スロット上端の境界線とほぼ重なる（タグ下端がスロット上端に食い込む/またがる形で配置）。タグ地色はくすんだ茶〜灰色がかった半透明、数字色は白。
6. 質感: スロット内部はレンガ/木目調の背景がそのまま透けて見える（スロット自体は透明で縁取りのみ）。光沢・グラデーションはほぼ無し、背景がそのまま透過している。
## cur
1. 線: スロット外形は上下左右1本ずつの明るいグレー線。番号タグ（6, 7）にも独立縁取り。
2. 枠の層: 単層。
3. 端の処理: フェードなし、均一。
4. 角: スロット本体はほぼ直角。番号タグは角丸。
5. 付属要素: 番号タグはスロット本体の真上だが、スロット上端との間に明確な隙間があり、タグはスロット外側に完全に浮いている（重なり・またがりなし）。タグ地色は濃い茶色〜黒に近い不透明、数字色は白。
6. 質感: スロット内部は濃い黒〜こげ茶のほぼ不透明な地色（背景の茶色は透けていない）。フラットな質感。
## 相違点
- 番号タグの配置がrefはスロット上端の境界線にまたがる/食い込む形、curはスロット上端から離れて完全に外側に浮く形で、明確に異なる。
- スロット内部の透過度がrefは背景（レンガ/木目）がそのまま透けて見えるのに対し、curはほぼ不透明な暗色で背景が透けない。
- タグ地色がrefは半透明のくすんだ茶灰色、curは不透明に近い濃茶〜黒。
- レシート画像を見ると、ref側の赤枠はスロット5とスロット6の境界をまたぐ形で、片方のスロットのみを過不足なく囲えていない（2スロット分の領域にまたがっている）。cur側の赤枠は番号タグ「6」からスロット上部の途中までしか囲っておらず、スロット下半分が枠外にはみ出している。
## レシートbox妥当性: NO（ref: 単一スロットに閉じず隣接スロットとの境界をまたいでいる／cur: スロット下部が枠に収まっておらず高さが不足している）

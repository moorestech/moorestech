# メイン充填スロット（アイテム入りスロット） 独立クロスチェック
## ref
1. 線: 上下左右とも1本ずつ、暗いグレーの単色線。内外の二重感はなし。
2. 枠の層: 単層（暗いグレーの縁取り1本のみ）。
3. 端の処理: フェードなし。四辺とも均一な太さ・濃さで端まで途切れず続く。
4. 角: ほぼ直角。わずかに丸みがある可能性はあるが目視ではほぼ直角に近い。
5. 付属要素: 個数「65」はスロット内側（アイテムアイコンに重なる形）の右下に配置。文字色は白、縁取り（アウトライン/ドロップシャドウ）が黒〜濃灰。背景チップ（色付き台座）は無く、テキスト単体に縁取りのみ。
6. 質感: スロット背景は白〜クリーム色のフラットな地色（模様なし）。アイコン（木の棒）は陰影・環境光・接地影のある3Dレンダー調の質感。光沢感は控えめ、透過は感じられない（背景が透けて見えない不透明な白地）。ベベルは枠自体には見られない。
## cur
1. 線: 上下左右とも1本ずつ、明るいグレー/白系の単色線。
2. 枠の層: 単層（明るいグレーの縁取り1本のみ）。
3. 端の処理: フェードなし。四辺とも均一で端まで一定。
4. 角: ほぼ直角（refと同程度、わずかな丸みの印象があるが大きな差ではない）。
5. 付属要素: 個数「10」はスロット内側の右下、アイコンに重なる形で配置。文字色はミントグリーン、縁取りは濃い色（黒〜濃灰）。背景チップなし、テキスト単体+アウトラインのみ（refと同構造）。
6. 質感: スロット背景は濃紺〜チャコールのフラットな地色。アイコン自体は上下2段の帯（明→暗のツートーン）で構成されたフラットな2Dスタイル化アイコンで、写実的な陰影レンダーではない。角に丸みのあるアイコン形状で軽いベベル的な段差表現はあるが、光沢・グラデーションは弱い。スロット周囲の背景にはうっすら菱形（ダイヤ柄）の装飾パターンが見える。
## 相違点
- スロット地色がrefは白/クリーム（不透明）、curは濃紺/チャコール。明暗が反転している。
- 個数テキストの色がrefは白、curはミントグリーンで異なる。
- アイコンの質感がrefは写実的3Dレンダー調、curはフラットな2段ツートーンのアイコン。
- 枠線の色がrefは暗色（明地に暗枠）、curは明色（暗地に明枠）で、コントラストの付け方が逆。
- cur側のみ、スロット周囲の背景に菱形パターンの装飾が視認できる（refには見えない）。
## レシートbox妥当性: YES（ref/curともに赤枠はスロット外形の四辺にほぼ一致し過不足は見られない）

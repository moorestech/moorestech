# ホットバー番号タグ（スロットの番号） 独立クロスチェック
## ref
- タグ位置: スロットの上端の境界線にまたがる形で配置。タグ下端がスロット上端の枠線と重なり合い、半分はスロット外・半分はスロット内（または枠線上）に食い込んでいるように見える。完全に外側に浮いているとは言えない。
- タグ地色: くすんだ茶色がかったグレー（半透明気味）。背景のレンガ模様がうっすら透けて見える印象。
- 数字色: 白。
- 枠: タグ自体にも薄いグレーの縁取りが1本あり、単層。
- 角: 小さめの角丸（直角に近いがわずかに丸い）。
## cur
- タグ位置: スロット本体から明確に離れて上に浮いている。タグ下端とスロット上端の間に視認できる隙間があり、完全にスロット外側に独立して配置。
- タグ地色: 濃い茶色〜黒に近い不透明色。背景は透けていない。
- 数字色: 白。
- 枠: 薄いグレーの縁取り1本、単層。
- 角: 角丸（refと同程度かやや丸みが強い印象）。
## 相違点
- 最大の違いはスロットとの位置関係: refはスロット上端の境界にまたがる/食い込む配置、curはスロットから完全に離れて浮く配置。
- タグ地色の透明度: refは半透明で背景が透け、curはほぼ不透明。
- 両者とも数字色は白で共通、枠も単層・角丸で構造自体は近い。
## レシートbox妥当性: 対象外（本要素にレシート画像の指定なし）

# 見出しオーナメント装飾（水平飾り線） 独立クロスチェック

## ref
1. 線は1本（単線）。中央の装飾部分で線自体は視覚的に途切れ、ダガー/矢じり形状に置き換わっている構成であり、「単線＋中央装飾」という1系統の意匠。
2. 端の処理: 「端まで均一（フェードなし）」（先頭-22%/末尾-22%、実質フェード無し）。
3. 角: 該当なし（矩形パネルではなく水平線装飾のため角の概念なし）。
4. 装飾: 中央に、内側を向く小さな矢じり(ダガー)形状が線の左右から中心に向かって配置され（► ◄ のように収束する形）、さらにその中心に二重構造の菱形（外側の菱形の中に一回り小さい菱形が入った「宝石」風の意匠）がある。ダガー部分の色はRGB(39,44,45)程度の暗いグレー、中心の菱形はRGB(57,61,67)とわずかに明るく、素材として一段階明るいハイライトが乗っている。
5. 該当テキストなし。背景はほぼ黒に近い暗いパネル上。
6. 質感: フラットだが中心の菱形にわずかな明暗の段差（ファセット風の光沢）があり、单なるベタ塗りではない印象。

## cur
1. 線は2本（近接する二重線、隙間あり）。中央部で二重線が途切れ、その隙間にダイヤ形が1つ収まる構成。
2. 端の処理: 「端まで均一（フェードなし）」。
3. 角: 該当なし。
4. 装飾: ダガー/矢じり形状は一切無い。中央には単色フラットなグレーの菱形（内部の階調やファセットは無い、ベタ塗りの単純な形）のみ。
5. 該当テキストなし。背景はダイヤ柄パネルではなく、素の草原(オリーブグリーン)がそのまま見えている（サンプルRGB(84,102,80)で緑寄り、他のcurパネル背景RGB(55,65,75)前後の青灰色とは明確に異なる＝この帯だけ不透明パネムの外/透過ゾーンにある）。
6. 質感: 完全にフラット。グラデーションやファセットの陰影は無い。

## 相違点
- 装飾モチーフが根本的に異なる: ref=矢じり(ダガー)が中心に収束+二重菱形の宝石。cur=装飾なしのプレーン単色菱形のみ。
- 線の本数: ref=単線、cur=二重線（隙間あり）。
- 背景: refは暗いパネル上、curは草原がそのまま透けるゾーンにあり、周囲の不透明ダイヤ柄パネルと質感が浮いている。
- cur側のレシート画像には、この二重線+ダイヤの意匠がアイテムスロット行の間の区切り線としても再利用されている様子が写っており（左右にスロットが並ぶ）、curでは同一モチーフが複数箇所で使い回されていることが確認できる。

## レシートbox妥当性: YES（ref/cur両方）
理由: 赤枠は線の始端〜終端にほぼ一致する形で水平方向を過不足なく覆っており、縦方向も線帯＋中央装飾を欠けなく収めている。

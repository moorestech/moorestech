# 中央パネル 独立クロスチェック
## ref
1. 線: クロップ上部に台形（ハンマーアイコンの乗ったタブ）の輪郭線、下部に別パネルの角に銀白の細い縁取りが1本見える。合計で複数の断片的な線。
2. 枠: 少なくとも2つの独立した枠らしき要素（上のタブの黒い縁取りと、下のパネルの銀白の縁取り）が層のように重なって見える。
3. 端の処理: いずれも端まで比較的くっきりしており、大きなフェードは見えない。
4. 角: 上のタブは斜めにカットされた台形状（左右が斜め線）。下のパネルの角は直角に近い。
5. 付属要素: ハンマーアイコンはタブの内側中央に配置。
6. 質感: タブはフラットなグレーで、アイコンは低コントラストの暗いオリーブ色でほぼ背景に溶け込んでいる。下のパネルは銀白の縁で立体感がある。

## cur
1. 線: 上辺・左辺に外周の縁取り線が1本ずつ見える（このクロップにはタブは写っていない）。
2. 枠: 単層。銀白のベベル枠で、光の当たり方で明暗のグラデーションがついている。
3. 端の処理: 端まで均一にくっきりと縁取られている。
4. 角: 左上角は直角（90度）に近い、鋭角的な角。丸みは感じられない。
5. 付属要素: このクロップ範囲には付属要素は写っていない。
6. 質感: フラットな緑がかったグレーの面に、銀白のベベル枠。光沢感のあるハイライトが枠の上辺・左辺に見える。

## 相違点
- refのクロップにはタブ（ハンマーアイコン付き台形）が含まれるが、curのクロップには写っていない（位置がずれている、もしくはcurにタブ自体が存在しない可能性）。
- refのタブは角が斜めカットの台形形状だが、curの対応パネル角は直角。
- refのアイコンは低彩度・低コントラストで視認しづらいが、curは（tab-top要素で確認した限り）色鮮やかな描き込みのアイコンになる傾向がある。

## レシートbox妥当性
ref: YES（緑の地面から暗いパネル内側への遷移に細い縦の境界線が見え、boxがその断面を貫いている）
cur: YES（同様に緑の地面から暗い内側への遷移をboxが貫いている。ただし境界線の色味がやや茶色がかっている点はrefと異なる）

# CRAFT RECIPEパネル 独立クロスチェック
## ref
1. 線: 見出し「CRAFT R...」の上下に細い水平線が1本ずつ、計2本のみ。
2. 枠: なし。パネルを囲む枠線は確認できない。
3. 端の処理: 左側で緑の地面から暗いオーバーレイへなだらかにフェードしており、明確な端がない。
4. 角: 角自体が存在しない。
5. 付属要素: 見出し上下の線はテキストの外側に位置。
6. 質感: フラットで透過感のある暗いオーバーレイ。ベベルなし。

## cur
1. 線: パネル外周の縁取りが上辺・左辺で見える（1本の連続線）。加えてクロップ左端にもう一つ別パネルの角の縁がわずかに写り込んでいる。
2. 枠: 単層。銀白のベベル枠で、外側が明るく内側にかけて陰影がつく。
3. 端の処理: 端まで均一にくっきりと縁取られている。
4. 角: 左上角は斜めにカットされた面取り角（直角でも丸みでもない）。
5. 付属要素: 見出し「CRAFT」はパネル内側のダイヤ柄背景の上に配置。
6. 質感: 暗いネイビー地にダイヤ（ひし形）柄のテクスチャ。枠はハイライト+シャドウで立体的なベベル感がある。

## 相違点
- refには枠・パネル背景が存在しない（見出し用の上下線のみ）が、curは銀白ベベル枠+ダイヤ柄の独立パネルとして描画されている（panel-invと同様の構造）。
- curの角は面取り（斜めカット）だが、refには角の概念がない。
- refは半透明で地面が透けるが、curは不透明な単色パネル。

## レシートbox妥当性
ref: NO（明確な縁が存在せず、boxは地面から暗いオーバーレイへのグラデーションを捉えているだけ。box右端にわずかにアイテムアイコンが見切れている程度）
cur: YES（boxが緑の地面→銀白の枠→パネル内側の順の断面を貫いている。box下部に斜めカット角の一部も入っている）

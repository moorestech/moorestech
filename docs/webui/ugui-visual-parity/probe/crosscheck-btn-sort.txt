# 整理ボタン 独立クロスチェック

## ref
1. 線: 該当なし。
2. 端の処理: 該当なし。エッジのコントラストが低く(輝度範囲49.5〜55.8と狭い)、くっきりした境界線としては検出されない＝縁がソフト。
3. 角: 数値計測は無いが、目視では直角に近い印象（丸みは確認できない）。
4. 装飾: なし。
5. 文字「整理」は白・太字。
6. 質感: 濃紺寄りの半透明な暗色矩形（サンプルRGB(89,93,103)、青みがかった暗色）。背景の草原がうっすら透けて見える程度の不透明度で、輪郭のはっきりした縁取りは無い。草原(ワールド)背景に直接乗っている。

## cur
1. 線: 該当なし。
2. 端の処理: エッジは輝度107.8→46.0へ約5pxで急落する非常にシャープな境界（明確な縁取り/くっきりした境界線あり）。
3. 角: 目視で明確な角丸（3ボタン中もっとも丸みが強い印象）。
4. 装飾: なし。
5. 文字「整理」は白・太字。
6. 質感: 不透明な濃いグレー〜黒に近い矩形に、はっきりしたライトグレーの縁取り線がある（サンプルRGB(128,128,128)は縁取り部分の色と見られ、中央の塗りはより暗い）。草原背景に乗っている点はrefと同じ。

## 相違点
- 縁の質感: refは半透明でソフトな縁（境界が曖昧）。curは不透明で縁取り線のはっきりしたシャープな境界。
- 角: refは直角寄り、curは明確な角丸で丸みの強さが異なる。
- 色味: refは青みがかった暗色、curはニュートラルなグレー〜黒で色相が異なる。
- 透過性: refは背景がうっすら透けるが、curは完全に不透明。

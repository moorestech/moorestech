# 中央見出しブロック（アイテム名タイトル） 独立クロスチェック

## ref
1. 線: タイトル文字の直上・直下には線が無い。文字の下、ボタンのさらに下にオーナメント線（単線＋中央装飾、詳細はornament項目参照）があるのみ。
2. 端の処理: N/A（このクロップ単体では対象外。ornament項目参照）。
3. 角: ボタン「レシピツリーで表示」の角は直角（別計測: corner-btn-tree-ref で radius≈0px と確定）。
4. 装飾: アイテムアイコンはこのクロップ範囲には写っていない（クロップがタイトル文字から始まっており、アイコンがあるかどうかは本クロップからは確認不能）。
5. タイトル文字は「石の斧」、白・太字、控えめな影付き。ボタン文字「レシピツリーで表示」は白〜薄グレー。
6. ボタンの質感: 濃い青からシアンへのグラデーション＋光沢のハイライト（右上に斜めの光の筋）があるグロッシー仕上げ。
7. 相違点は下記参照。

## cur
1. 線: タイトル文字の直上・直下にも線は無い。文字の上にアイテムアイコン（ハンマーの絵柄が入った暗色タイル）が別要素として存在する。ボタンの下にオーナメント線（二重線＋プレーンなダイヤ、詳細はornament項目参照）がある。
2. 端の処理: N/A（ornament項目参照）。
3. 角: ボタンの角は明確な角丸（別計測: corner-btn-tree-cur で radius≈17px と確定）。アイコンタイルの角は直角に近い印象（数値未計測）。
4. 装飾: タイトル文字の上にアイテムアイコン（ハンマー）タイルが表示されている。
5. タイトル文字が「石の斧」ではなく「Item 100」というプレースホルダー風の英語表記になっている（未翻訳/未反映の疑い）。フォントは白〜薄グレーの太字でrefと近いが、内容が異なる。
6. ボタンの質感: 単色フラットな中間青、光沢・グラデーションなし。

## 相違点
- ボタン角: ref=直角(0px) / cur=角丸(17px) と明確に異なる（測定済み）。
- ボタン質感: ref=グロッシーなグラデーション+ハイライト / cur=フラット単色。
- タイトル直上のアイコン: cur側にのみアイテムアイコンタイルが付加されている（ref側はクロップ範囲外で有無不明）。
- タイトル文言: refは実アイテム名（石の斧）、curはプレースホルダー的な「Item 100」表記で、ローカライズ/データ反映のcontent差がある。
- オーナメント線の構成差はornament項目を参照（本要素の直下に位置するため連動する差異）。

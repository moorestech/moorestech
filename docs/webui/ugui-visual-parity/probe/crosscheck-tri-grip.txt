# 三角グリップ装飾（パネル右下の三角形） 独立クロスチェック
## ref
1. 線: 枠線が上辺・右辺に見え、二重線（外側の太い線+内側にもう1本細い線）のように見える。
2. 枠: 2層（太めの外枠+その内側にやや細い縁）。
3. 端の処理: 端まで均一にくっきりしており、フェードはない。
4. 角: 直角（90度）の鋭い角。
5. 付属要素: 角の内側（枠の内角部分）に小さな白っぽい三角形（斜めハッチ柄）の装飾が付いている。付属要素は枠の内側に位置。
6. 質感: 背景はレンガ状のテクスチャ。枠自体はフラットでグレー系、光沢は弱い。三角形装飾も単色フラットに近い。

## cur
1. 線: 枠線が二重線（外側の縁+内側にもう1本）のように見える。加えて左上から別要素（青い角丸の縁）が重なって写り込んでいる。
2. 枠: 2層的に見える（グレーがかった外枠+内側にもう1段）。
3. 端の処理: 端まで均一にくっきりしている。
4. 角: 丸角（明確な半径を持つ丸み）。refの直角とは異なる。
5. 付属要素: 三角形の装飾は見当たらない。角の内側は無地で装飾なし。
6. 質感: 背景は茶色のグラデーション（レンガ的テクスチャではない、より単調な色面）。枠はフラットでグレーがかった色。

## 相違点
- refには枠の内角に三角形（グリップ）装飾があるが、curには同等の装飾が見当たらない（欠落している）。
- refの角は直角だが、curの角は丸角。
- refの背景はレンガの質感テクスチャが見えるが、curの背景はよりのっぺりしたグラデーションで質感の情報量が少ない。
- curには青い角丸要素が左上から重なって写り込んでおり、refには存在しない別レイヤーが視界に入っている。

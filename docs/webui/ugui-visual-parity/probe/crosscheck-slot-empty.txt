# メイン空スロット 独立クロスチェック
## ref
1. 線: 上下左右とも1本ずつ、明るめのグレー系単色線。
2. 枠の層: 単層。
3. 端の処理: フェードなし、四辺均一。
4. 角: ほぼ直角（大きな丸みなし）。
5. 付属要素: なし（数字・タグとも無し、空スロットのため）。
6. 質感: 地色は暗いスレートブルー〜グレーの半透明的な色味。内部にうっすら斜めの菱形/ジグザグ模様が透けて見える（背景パネルの装飾パターンが透過して見えている）。光沢・グラデーションは弱くほぼフラット。
## cur
1. 線: 上下左右とも1本ずつ、明るいグレー/白系単色線。
2. 枠の層: 単層。
3. 端の処理: フェードなし、四辺均一。
4. 角: ほぼ直角（refとほぼ同程度）。
5. 付属要素: なし。
6. 質感: 地色は濃紺〜チャコールに近い暗色で、内部にうっすら斜めの菱形模様が透けて見える（refと同様に背景パターンが透過）。フラットな質感でグラデーション・光沢はほぼ無し。
## 相違点
- 地色の色味がrefはスレートブルー寄りのグレー、curはより黒に近いチャコールで、僅かに色相が異なる（refの方がやや青みが強い）。
- 両者とも背景パターンが透けて見える点は共通（充填スロットのref（不透明白）とは対照的に、空スロットは両バージョンとも半透明で背景パターンが見える）。
- 枠線の色はref/curともに近い明るさのグレーで大きな差はない。
- refの空スロットの地色はスロット充填時（白/クリーム）と明確に異なる色を使うのに対し、curは充填時（濃紺/チャコール）とほぼ同じ地色を空スロットにも使っており、状態による地色の作り分けの有無がref/curで異なる。
## レシートbox妥当性: YES（ref/curともに赤枠はスロット外形の四辺に一致しており過不足なし）

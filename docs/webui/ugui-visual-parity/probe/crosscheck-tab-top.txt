# 中央パネル上部タブ（ハンマーアイコンのタブ） 独立クロスチェック
## ref
1. 線: タブの輪郭線（黒っぽい縁）が上辺・左右斜辺に見える。下のパネルとの境目には銀白の細い線が別途1本。
2. 枠: 2層的に見える。タブ自体の暗い縁取りと、下に重なるパネルの銀白縁取りが別々の層として存在。
3. 端の処理: 端まで比較的くっきりしているが、タブ全体がやや暗い黒縁でぼかし気味の輪郭を持つ。
4. 角: 台形（左右が斜めにカットされた形状）。直角でも丸角でもない、斜辺による面取り形状。
5. 付属要素: ハンマーアイコンはタブの内側中央に配置。
6. 質感: フラットなグレー。アイコンは低彩度・低コントラストの暗いオリーブ単色シルエットで、背景とほぼ同化して見にくい。ベベル感は薄い。

## cur
1. 線: タブ自体の輪郭線は明確な1本の縁取り。タブ下部でパネル上辺の白い線と重なっている。
2. 枠: 単層。角丸の縁取りが1本。
3. 端の処理: 端まで均一にくっきりと縁取られている。
4. 角: 丸角（四隅すべて丸い半径を持つ）。
5. 付属要素: ハンマーアイコンはタブ内側中央。加えてタブの下・パネル外側にテキスト「tem 100」（Item 100の一部）が見切れており、これはタブの外側に位置する付属要素。
6. 質感: フラットな濃紺地に、金属光沢のあるディテール描き込みのハンマーアイコン（グラデーション・ハイライトあり）。タブの縁は単色でベベル感は薄いが、アイコン自体に立体感がある。

## 相違点
- refのタブは台形（斜辺カット）だが、curのタブは角丸長方形。形状が根本的に異なる。
- refのアイコンは低彩度・低コントラストで背景に溶け込むが、curのアイコンは色鮮やかで立体的な描き込み（メタリックなハイライト）がありはっきり見える。
- curにはタブの外側に「Item 100」という個数表示テキストが付随しているが、refのクロップにはそのようなテキストは見えない。
- refはタブと下パネルの間に別レイヤーの縁取りが見えるが、curはタブと下のパネル境界線が重なって一体的に見える。

# レシピグリッドスロット 独立クロスチェック
## ref
1. 線: 上下左右とも1本ずつ、暗い単色線。ハイライト枠などの追加線は無し。
2. 枠の層: 単層のみ（暗い縁取り1本）。
3. 端の処理: フェードなし、四辺均一。
4. 角: ほぼ直角。
5. 付属要素: 個数表示や番号タグは見当たらない（このクロップ範囲では無し）。
6. 質感: 地色は淡いグレー（メインスロットの白/クリームより少しグレーがかった色）。アイコン（石を積んだような造形物）は陰影・質感のある写実的な3Dレンダー調。光沢感はやや控えめ、マットに近い。上部にスロットとは別に薄い水色〜白の細い横線（区切り線）が見える。
## cur
1. 線: スロット自体の内側枠（暗色・単層・直角に近い角）に加え、外側にシアン（水色）の縁取りリングが1本追加されている。上部には細い横線が2本並んで見える（区切り線が二重）。
2. 枠の層: 二層構造。内側=通常スロットと同じ暗い縁取り、外側=シアンの太めのリング（角に丸みあり）。
3. 端の処理: 内側の暗枠はフェードなし均一。外側のシアンリングは輪郭がやや発光的（にじみ/グロー）に見え、輪郭部分がわずかに柔らかく広がる印象。
4. 角: 内側の暗枠はほぼ直角、外側のシアンリングは明確に角丸（半径が大きめ）。
5. 付属要素: 数字等は見当たらない。
6. 質感: 地色は濃い黒に近いダーク色。アイコン（オレンジ系の四角、上下2段のツートーン）はフラットな2Dスタイルで、refのような写実レンダーではない。シアンリングには光るような質感（グロー/発光感）があり、ベベルというよりネオン的な縁取り。
## 相違点
- curのみ、外側にシアンの角丸ハイライトリング（グロー付き）が存在し、refには同様の強調枠が見られない。
- refの枠は単層で直角、curの枠は二層（直角の内枠＋角丸の外側リング）という構造差。
- アイコンの質感がrefは写実3Dレンダー、curはフラットな2段ツートーン（他スロットと同傾向）。
- 地色がrefは淡いグレー、curは黒に近いダーク色で明暗が逆。
- 上部の区切り線がrefは1本、curは2本（他の見出し部でも見られる二重線パターンと一致）。

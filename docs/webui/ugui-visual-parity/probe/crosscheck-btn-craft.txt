# CRAFTボタン 独立クロスチェック

## ref
1. 線: 該当なし。ボタン下方にパネル区切りの横線が別途見える程度。
2. 端の処理: 該当なし。
3. 角: 直角（radius≈0px、corner-btn-craft-ref計測で確定）。
4. 装飾: なし。上辺にごく細い明るい縁取りライン。
5. 文字「CRAFT」は白・太字、文字間がやや広め。
6. 質感: 濃紺〜シアンのグラデーション＋光沢ハイライト(グロッシー)。背景はレンガ壁のテクスチャで、ボタンはその上に浮いている。
幅の相対感: ボタン幅はクロップ(パネル)幅の半分弱程度で、左右にレンガ背景の余白が大きく残る＝パネル幅に対して控えめな幅。

## cur
1. 線: 該当なし。
2. 端の処理: 該当なし。
3. 角: 角丸（radius≈4px、corner-btn-craft-cur計測で確定。refよりは弱いがrefの直角に対しては明確に丸い）。
4. 装飾: なし。
5. 文字「CRAFT」は白・太字。
6. 質感: 単色フラットな中間青。背景はレンガではなく、境界線で囲われた木目調の茶色グラデーションパネル。
幅の相対感: ボタンはパネルの内側パディングぎりぎりまで広がっており、パネル幅に対してほぼ全幅（9割以上）を占める＝refより明らかに横幅が広い。

## 相違点
- 角: ref=直角(0px) / cur=角丸(4px)。
- 質感: ref=グラデーション+グロス / cur=フラット単色。
- 背景: ref=レンガ壁テクスチャがそのまま見える / curは境界線付きの木目調パネルで背景処理が異なる。
- 幅: refはパネル幅に対して控えめ(半分弱)なのに対し、curはパネル幅いっぱいまで広がっており、パネル内での占有率が大きく異なる。

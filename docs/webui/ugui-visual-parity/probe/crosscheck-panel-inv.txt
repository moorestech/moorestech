# 持ち物パネル 独立クロスチェック
## ref
1. 線: 見出しテキスト「持ち物」の上下に細い水平線が1本ずつ、計2本のみ。左右・パネル外周に線は見えない。
2. 枠: なし。パネル自体を区切る枠線は確認できず、背景が緑の地面テクスチャからやや暗いオーバーレイへ滲むように変化しているだけ。
3. 端の処理: 明確な「端」が存在しない。暗いオーバーレイは輪郭を持たずグラデーションで地面に溶け込んでいる（フェード的）。
4. 角: 角自体が存在しない（枠がないため角の形状を評価できない）。
5. 付属要素: 見出し上下の線はテキストの外側（上と下）に位置する。
6. 質感: フラット。光沢・グラデ・透過感のある暗いレイヤーのみで、立体的なベベルは見えない。

## cur
1. 線: パネル外周の枠線として上辺・左辺が見え（クロップ範囲内）、外側1本の縁取りのみ。内側に別線は無い。
2. 枠: 単層。ただし外周にライト（明るい銀白）→内側にかけて暗くなるグラデーションがあり、断面としては1本だが立体的な陰影がついている。
3. 端の処理: 端まで均一な太さ・明るさで縁取られており、フェードはしない（枠線としてくっきり）。
4. 角: 左上角は直角ではなく斜めにカットされた面取り（チャンファー）角。半径のある丸みではなく直線的な斜めカット。
5. 付属要素: 見出し「持ち物」はパネル内側（ダイヤ柄の暗い背景の上）に配置。
6. 質感: フラットな暗いネイビー地に、規則的なダイヤ（ひし形）柄のテクスチャ。枠自体はハイライト+シャドウで立体的なベベル感がある。パネル面自体は光沢なし。

## 相違点
- refには枠線・背景パネルが存在しない（見出し用の上下線のみ）のに対し、curは明確な銀白ベベル枠+ダイヤ柄背景を持つ独立したパネルとして描画されている。
- refの装飾は「テキストの上下の線」だけ、curは「パネル全体を囲む枠」という全く異なる構造。
- curの角は斜めカット（面取り）だが、refにはそもそも角の概念がない。
- refは背景に地面テクスチャが透けて見える半透明的表現、curは不透明な単色パネル。

## レシートbox妥当性
ref: NO（対象の縁が存在しないため、boxは地面のグラデーションを捉えているだけで「パネル縁」を貫通していない）
cur: YES（boxが緑の地面→銀白の枠→パネル内側の順に明確な断面を貫いている）
